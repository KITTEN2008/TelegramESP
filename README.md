# TelegramESP
TelegramESP - это библиотека для ESP32 для работы с Telegram-ботами, которая позволит вам работать с токеном бота и получать от него сообщения, отправлять ответы и в общем развивать IoT для ESP32!

| Команда  | Значение |
| ------------- | ------------- |
| #include <TelegramESP.h>  | Подключение библиотеки  |
